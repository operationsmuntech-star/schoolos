### Why `staticfiles/` is in .gitignore

```
.gitignore line 38: staticfiles/

Reason:
- staticfiles/ is GENERATED by Django's collectstatic command
- Copying it to Git wastes space and causes merge conflicts
- Railway always regenerates it during deployment
- Local staticfiles/ copy is only for testing locally
```

### Static Files Resolution

**Development (DEBUG=True):**
```

```

**Production (DEBUG=False, Railway):**
```
Browser requests: /static/css/auth.css
WhiteNoise middleware serves from: staticfiles/css/auth.css
Source: static/css/auth.css (copied by collectstatic)
```
